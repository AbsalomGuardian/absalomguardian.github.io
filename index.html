Tic-tac-toe AI
Programming Abstractions: Python, Fall 2025

This is a program that simulates Tic-tac-toe entirely in the console, using ASCII visuals. The user selects the settings while the program is running and can choose between player vs player, player vs AI, and AI vs AI. Players can also set their own name. The AI has three different algorithms, random, min-max, and using the solved algorithm for tic-tac-toe. The last one will always win or tie. The algorithm is completely hand made, and both it and the minmax algorithm are able to do many assessments of the board per turn in a timely manner.

Graph ADT
Introduction to Data Structures and Algorithms, Winter 2025

In my algorithms class, I made several ADTs- advanced datatypes- in C or C++. Working without any complex libraries, I implemented the specified functions and typically wrote a script that utilized the datatype as well. Below are some selected examples, usually the final example in a series of assignments that iterated on the previous one. All of the scripts are controlled by writing in a .txt file and then output is saved in a separate .txt file.


A Graph ADT, based on a previously made doubly linked list ADT. The graph lines are unweighted and undirected. Based on an input .txt file, it will assemble a graph and then use DFS search to determine which strongly connected components the graph has and then print that information to a separate .txt file.

Square Matrix ADT
Algorithms, Witner 2025

A square matrix ADT, which can preform scalar multiplication, addition, subtraction, transpose and product operations. These operations are done in an efficient manner. The matrix consists of a 1d array of Lists (the previously created ADT), and only non-zero entries are stored.

BigInteger ADT
Introduction to Data Structures and Algorithms, Winter 2025

A BigInteger ADT, which stores its value as a list of Long variables. These lists can be summed or subtracted together; or multiplied by a scalar. And the final result is sure to be normalized, with the Long values remaining properly spaced out despite any carries that might have occurred. The client module of this program was a calculator that preformed a specified list of functions on two given big integers.

Dictionary ADT
Introduction to Data Structures and Algorithms, Winter 2025

A dictionary ADT, which organizes its contents as a binary tree. It stores keys which are English words, and its value is the number of times a specific word has been counted in the input text. The script for this assignment would return the word frequency of an input text. The binary tree was organized by the alphabetical ordering of the keys.
